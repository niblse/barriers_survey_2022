# barriers_survey_2022
Revisiting Barriers to Implementation of Bioinformatics into Life Sciences Education

# Table of Contents:
**Merged_Data_Anonymous.csv** contains survey data after removing selected identifier columns to ensure anonymity of participants.

**NIBLSESurvey2DataAnalysis_All.Rmd** and **NIBLSESurvey2DataAnalysis_All.docx** are R markdown and Word documents with all analyses performed of the anonymized data set.

**figure_.R** files are the R scripts used to generate the figures in the manuscript.
